Ты — эксперт по Telegram-ботам и веб-автоматизации. Напиши **полный рабочий проект** Telegram-бота для пожилого отца в Новогрудке.

Требования:
- Python + aiogram 3.x (async)
- Playwright (async_api) для автоматизации
- FSMContext для состояния (restaurant, cart, category)
- BeautifulSoup4 + aiohttp для парсинга меню (чтобы бот показывал актуальное меню кнопками)
- Развёртывание на Railway.app (Hobby план)

Сайты доставки:
1. https://vkusnovmeste.by/
2. https://sadypobedy.by/

Бот работает ТОЛЬКО кнопками (отец не вводит текст):
1. /start → ReplyKeyboard с 2 кнопками: «Вкусно вместе» и «Сады Победы»
2. Выбор ресторана → парсим категории (используй Playwright page.get_by_text или BeautifulSoup) → InlineKeyboard с категориями
3. Выбор категории → парсим блюда (название + цена) → показываем как InlineKeyboard (callback_data = "add|НАЗВАНИЕ|ЦЕНА")
4. Корзина в памяти (dict): кнопки «+1», «-1», «Корзина», «Очистить», «Оформить заказ»
5. «Оформить заказ»:
   - Playwright открывает сайт выбранного ресторана
   - Для каждой позиции в корзине: ищем по точному тексту get_by_text(dish_name) и кликаем кнопку "Добавить" / "В корзину"
   - Переходим в корзину (ищем "Корзина" или иконку)
   - Заполняем фиксированные данные:
     • Телефон: +375XXXXXXXXX
     • Адрес: "г. Новогрудок, ул. ______, дом __, кв. __"
     • Комментарий: "Перезвонить для уточнения заказа"
     • Оплата: "Наличными курьеру"
   - Кликаем "Оформить заказ"
   - Делаем screenshot страницы подтверждения и отправляем отцу + текст "✅ Заказ оформлен! Курьер скоро перезвонит."

Дополнительно:
